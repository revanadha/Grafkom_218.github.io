Final Project Grafika Komputer

Implementasi 3 materi: Linear, Transformasi 2D, dan Color Space. Tanpa library eksternal.

Identitas Program

Program ini dibuat untuk memenuhi tugas Final Project Grafika Komputer Semester Genap 2025/2026. Tiga materi yang dipilih adalah: A. Linear B. Transformasi C. Color Space

Struktur program: Input pengguna → Proses algoritma → Output visual dan numerik.

A. Linear: DDA, Bresenham, dan Midpoint Circle

Bagian ini mengimplementasikan algoritma dasar pembentukan objek grafis berbasis titik/piksel. DDA dan Bresenham digunakan untuk membentuk garis, sedangkan Midpoint digunakan untuk membentuk lingkaran.

Input Garis

Input Lingkaran

Koordinat kartesius divisualisasikan pada canvas. Satu kotak grid = 20 piksel.

B. Transformasi 2D: Translasi, Skala, Rotasi, Refleksi, dan Shearing

Bagian ini mengimplementasikan transformasi objek 2D pada sebuah segitiga. Setiap titik objek diproses menggunakan rumus transformasi matriks 2D.

Input Titik Segitiga

Jenis Transformasi

Objek awal digambar dengan garis putus-putus, objek hasil transformasi digambar dengan garis penuh.

C. Color Space: RGB, CMYK, dan HSV

Bagian ini mengimplementasikan konversi ruang warna dari RGB ke CMYK dan HSV. Input warna RGB diproses secara matematis, lalu ditampilkan dalam bentuk nilai dan pratinjau warna.

Input RGB

Hasil Konversi

Ruang Warna Nilai Keterangan

Rumus CMYK menggunakan normalisasi RGB ke rentang 0-1. Rumus HSV menggunakan nilai maksimum, minimum, dan selisih kanal warna.
